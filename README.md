

# 🗳️ E-Voting Web Application

## 📌 Overview

This project is a **secure and user-friendly electronic voting system** designed to streamline  voting process. The application allows registered users to cast their votes digitally, while administrators can manage elections, candidates, and view results in real-time.

The goal is to provide a transparent, efficient, and accessible alternative to traditional paper-based voting systems.

---

## 🚀 Features

* 🔐 **User Authentication** – Secure login and registration for voters.
* 🗳️ **Cast Vote** – Each voter can vote only once.
* 👤 **Candidate Management** – Admins can add, edit, and remove candidates.
* 📊 **Real-Time Results** – Dynamic results display as votes are cast.
* 📅 **Election Management** – Admins can create and manage elections.
* 📱 **Responsive Design** – Works across devices (desktop, tablet, mobile).

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP 
* **Database:** MySQL 
* **Other Tools:** AJAX, Bootstrap
